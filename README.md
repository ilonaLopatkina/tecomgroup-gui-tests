# Автотесты GUI для tecomgroup.ru

Автоматизированное тестирование пользовательского интерфейса сайта [tecomgroup.ru](https://tecomgroup.ru).  
Проект выполнен в рамках **34-го Всероссийского конкурса студентов вузов по графическим информационным технологиям** (НГТУ им. Р.Е. Алексеева).

---

## Стек технологий

| Технология | Версия |
|------------|--------|
| Java | 21 |
| JUnit Jupiter | 6.0.1 |
| Playwright | 1.56.0 |
| Maven Surefire Plugin | 3.2.2 |

---

## Что покрыто тестами

- **Валидация UI-элементов** — проверка видимости заголовков, текстовых блоков, изображений, иконок и кнопок на всех ключевых страницах
- **Навигация** — работа выпадающих меню, переходы между страницами
- **Формы** — заполнение и отправка форм обратной связи, подписки на новости, заявок на консультацию (позитивные и негативные сценарии)
- **Поиск** — проверка поиска по сайту и по новостям, обработка пустых и некорректных запросов
- **Внешние переходы** — открытие PDF-документов, переходы на внешние сайты, открытие страниц в новых вкладках
- **Интерактивные элементы** — раскрывающиеся блоки, кликабельные карточки продуктов и проектов, кнопки с иконками
